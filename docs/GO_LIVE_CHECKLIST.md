# Go-Live Checklist (Internal)

Use this checklist before enabling the platform for all ~500 employees.

## Pilot (recommended)
- [ ] Departments created and verified (EN/AR)
- [ ] 20–50 users uploaded (1–2 departments)
- [ ] Templates reviewed (default + anonymity rules)
- [ ] Send a pilot evaluation cycle
- [ ] Confirm:
  - [ ] Users can open the evaluation from notifications
  - [ ] Submissions work (anonymous and non-anonymous)
  - [ ] Dashboards show data after completion
  - [ ] Export Center produces Excel backups

## Scale-up
- [ ] Full company roster imported (500 users)
- [ ] Roles verified (Admin/Audit/Super User/User)
- [ ] Permissions verified (who can see what)
- [ ] Admin checklist completed
- [ ] Weekly export backup schedule assigned (manual or automation)

## Operations
- [ ] Support contact defined (Admin)
- [ ] Written communication sent to employees (deadline + expectations)
- [ ] First full evaluation cycle scheduled
